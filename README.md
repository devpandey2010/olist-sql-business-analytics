## 🚀 Project Overview

This repository contains **100 real-world SQL practice questions** built on the **Olist Brazilian E-Commerce Dataset**.

The goal is to practice SQL **the way real Data Analysts, Business Analysts, and Product Analysts work in large companies** — not just toy queries.

Each question simulates **actual business problems** faced by e-commerce platforms like Amazon, Flipkart, or Shopify.

---

## 🎯 Aim of This Project

✔ Strengthen SQL querying skills  
✔ Learn real-world business and analytics terminology  
✔ Practice KPI building and reporting  
✔ Develop problem-solving ability for analytics interviews  
✔ Build portfolio-ready project experience  

---

## 📘 What You Will Learn

| Area | Concepts Learned |
|------|------------------|
| SQL Fundamentals | Joins, Aggregations, Filtering |
| Business Analytics | Revenue, AOV, Growth, KPI Reporting |
| Product Analytics | Retention, Funnels, Cohorts |
| Customer Analytics | Repeat Rate, LTV, Segmentation |
| Supply Chain Analytics | Delivery SLA, Logistics Performance |
| Advanced SQL | CTEs, Window Functions, Subqueries |

---

## 💡 Why This is Different from Typical SQL Practice

Most SQL practice datasets teach **syntax only**.

This project teaches:
> **SQL + Business Thinking + Analytics Mindset**

Each question:
- Uses real business terminology
- Focuses on decision-making insights
- Mirrors interview questions from top tech companies
- Builds storytelling with data
